# Security Policy

## Reporting a Vulnerability

If you discover a security issue related to this portfolio site, please do **not** open a public issue.

Please report it privately using GitHub private vulnerability reporting, or contact:

**[security@joshuacmcdonald.com](mailto:security@joshuacmcdonald.com)**

If that address is not active yet, use:

**[contact@joshuacmcdonald.com](mailto:contact@joshuacmcdonald.com)**

Please include:

* A description of the issue
* The affected page, file, or URL
* Steps to reproduce the issue
* Any screenshots or supporting details
* Any recommended fix, if available

## Scope

This policy applies only to this public portfolio repository and the static website files contained here.

Do not test against systems, accounts, infrastructure, cloud resources, networks, or services that are not part of this public portfolio site.

## Public-Safe Content

This repository is intended to contain only public-safe portfolio material. Internal IP addresses, credentials, private screenshots, ticket numbers, confidential employer information, and sensitive infrastructure details should not be committed.

## Response

I will review reports as soon as reasonably possible and take appropriate action if the issue is valid.
